📖 วิธีใช้งาน
  1. สร้าง App ที่ developers.facebook.com
  2. เปิด Graph API Explorer → เลือก Page Token
  3. เพิ่มสิทธิ์: pages_manage_posts, pages_read_engagement
  4. ค้นหาจากเพจ → เลือกโพสต์/โฆษณา/รูปภาพ เพิ่มเข้าคิว
  5. เลือกรูปจากเครื่อง → กดเริ่ม (ระบบทำทีละโพสต์)
💡 หา Post ID:
1. เปิดโพสต์/รูป → คลิกขวาที่วันที่ → Copy Link
2. วาง URL ลงมาได้เลย ระบบแปลงให้อัตโนมัติ
URL ที่รองรับ:
facebook.com/.../posts/123456
facebook.com/photo?fbid=123456
facebook.com/.../photos/a.xxx/123456
